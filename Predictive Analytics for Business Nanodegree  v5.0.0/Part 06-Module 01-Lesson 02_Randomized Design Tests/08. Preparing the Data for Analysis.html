Workflow Steps

Workflow Steps

  • Use the Input tool to import the grocery website A/B test data
  • Add a Browse tool and then run the workflow to take a look at the data
  • Use the Select tool to convert the visit page flag to type int16
  • Use the Summarize tool to group data by IP address, server ID, and the logged in flag, and sum the visit page flag field
  • Use the Formula tool to set up a formula such that “if the sum of the visit flag field != 0, then 1, otherwise 0”
  • Within the formula tool, create a new string field called group, with a formula such that “if server ID = 1, then treatment, otherwise control”
  • Use the Filter tool to filter for Logged in flag != 1
  • Add a Browse tool to the T output, and run the workflow
