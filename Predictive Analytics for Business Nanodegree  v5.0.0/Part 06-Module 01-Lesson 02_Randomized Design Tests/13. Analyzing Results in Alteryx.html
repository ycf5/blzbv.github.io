13. Analyzing Results in Alteryx

Analyzing Results in Alteryx

Workflow Steps

Workflow Steps

  • Continuing from the workflow in the “Preparing the Data for Analysis” demo, add a Test of Means tool to the data stream
    • Select visit page flag for the response field and group as the group identifier
  • Run the workflow and take a look at the resulting p-value
  • Use the Summarize tool to count the number of visits, grouped by treatment or control group and whether or not they clicked on a link
  • Run the workflow
  • Add and configure a Cross-Tab tool:
    • Select group for the value to group data
    • Select visit flag for the new column headers
    • Select count for the value for new columns
    • Select sum and percent row for the method for aggregating values
  • Run the workflow again to see the results
