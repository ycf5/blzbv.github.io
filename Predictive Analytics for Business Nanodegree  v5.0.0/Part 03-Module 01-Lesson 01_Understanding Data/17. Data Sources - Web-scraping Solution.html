17. Data Sources - Web-scraping Solution

Solution

Step 1:


Start with a text input tool, enter the URL of the page and URL as the field name.

Step 2:

Then we connect the download tool and add a browse tool. Run the workflow to see the results in the DownloadData cell. This cell contains the entire web page.

Step 3:

To output this to a CSV file, connect the Select tool first to remove the two extraneous fields that do not contain the data from the web page: URL and DownloadHeaders.

Step 4:

Use the Output data tool, change the file format to CSV, enter '\0' in the Delimiter option to denote no delimiter and make sure that the option denoting first row contains field names is deselected. Run the workflow.
