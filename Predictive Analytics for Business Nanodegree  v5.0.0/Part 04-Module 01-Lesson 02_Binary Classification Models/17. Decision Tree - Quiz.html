17. Decision Tree - Quiz

Steps

Prepare for the Quiz

Before beginning the quiz, build out the same decision tree model that you just observed.

  • First, use an Input Data tool to bring in the Hotel Loyalty Data file
  • Second, use a Select tool to set the data types of each variable
  • Third, use a Create Samples tool and select 70% as the estimation percentage
  • Fourth, attach a Decision Tree tool and select the variables to be used in the model
  • Fifth, attach Browse tools to both the R and I output nodes

Use the results found in the browse tools to answer the following questions.

Q1

What are the 2 most important variables to the model?
SOLUTION:
  • Spend
  • Stays.per.year

Q2

When classifying all Yes's against the Cross Validation sample, what is the overall accuracy?
SOLUTION: 68%
