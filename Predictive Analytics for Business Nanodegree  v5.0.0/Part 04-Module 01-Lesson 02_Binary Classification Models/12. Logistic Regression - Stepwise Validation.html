12. Logistic Regression - Stepwise Validation

Logistic Regression - Stepwise Validation

summary

Key Summary

Note that the Stepwise Regression tool is a tool to help you reduce and figure out which predictor variables have a good chance of being in the model, but it is not a tool that can automatically find all of the appropriate predictor variables in one run.

There is still a lot of work that needs to get done to explore the predictor variables that the Stepwise Regression tool gives you. The Stepwise Regression tool will speed up the process for you in choosing predictor variables.

Download

Installing the Model Comparison Tool

  1. Go to https://gallery.alteryx.com and sign in to your account. If you do not have a Sign In create one via the Sign Up section
  2. Once signed in go to the District portion of the Alteryx Gallery and click on Predictive District
  3. Once inside of the Predictive District go to the Model Comparison tool
  4. Download the file and open it up
  5. When you open it up Alteryx will open as well asking you to import a workflow package; agree to this and say Import. This will download the information you need.
  6. Once the workflow/macro opens, Run the workflow and hit Save.
  7. Go to Options -> User Settings -> Edit User Settings -> Macros, add an entry there (click the + button)
  8. Click the … button next to search path and select the Model Comparison file you downloaded. Click Apply.
  9. Close Alteryx and open a new instance of Alteryx. In the Predictive Section you should see the new Model Comparison tool.

rocr error

ROCR error

When running the Model Comparison Tool, if you get stuck with an error that says Error in install.packages("ROCR", repos = repo), there is a permission error in your computer that is preventing Alteryx from installing an R package. This means that Alteryx needs to download additional files in the R system to run this Model Comparison tool in Alteryx, but cannot do it.
