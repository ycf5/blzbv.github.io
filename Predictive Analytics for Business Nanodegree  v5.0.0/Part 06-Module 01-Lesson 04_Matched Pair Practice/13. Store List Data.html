13. Store List Data

Store List Data

Workflow Steps

Workflow Steps

  • Using the same workflow from the “Weekly Store Traffic Data” demo, add a Filter tool to remove transactions that do not include a facial
  • Connect the filter to the output of the Summarize tool
  • Filter for facial flag = 1
  • Use the Summarize tool and group by region and store to get a list of stores in each region
  • Use the Input tool to import the file containing the information about which stores were designated as treatment stores
  • Use the Select tool to deselect price level and rename the test price level to “test group”
  • Use the Join tool to add the treatment store data to the main data stream
  • Use the Formula tool on the L output of the join to identify control candidate stores
  • After dragging and connecting the Formula tool, create a new field with the same name and type as the test group field from the J output of the join
  • Type in CC for the expression to add the field test group to the data with a value of CC for all the control stores
  • Use the Union tool to combine the two output tables of the join
  • Drag an Output tool onto the canvas, and save the file for later use
