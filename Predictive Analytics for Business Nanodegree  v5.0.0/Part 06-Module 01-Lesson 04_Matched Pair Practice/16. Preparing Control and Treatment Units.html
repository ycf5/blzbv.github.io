16. Preparing Control and Treatment Units

Preparing Control Treatment Units Part 1

Workflow Steps

Workflow Steps Part 1

  • Use the Input tool to import the weekly store traffic data
  • Drag an A/B Trend tool onto the canvas and connect it to the data stream
  • Configure the A/B Trend tool:
    • Select store as the unit identifier
    • Select week end as the reporting period date
    • Select count as the performance measure
    • Select weekly as the report period type
    • Select 6 as the number of periods
    • Select the test start date (March 20, 2014)
  • Run the workflow

Workflow Part 1

Preparing Control Treatment Units Part 2

Workflow Steps Part 2

Workflow Steps Part 2

  • Use the Input tool to import the store list data
  • Use the Join tool to connect the weekly traffic data to the store list data
  • Join using the store field, and deselect the right store input to avoid duplicate columns in the data stream
  • Use the Filter tool to filter for region = midwest
  • Use a new Filter tool to filter for test group != CC, so the T output of the filter will stream the treatment data and the F output will stream the control data
  • Add an A/B Controls tool to the canvas and connect the data with the treatment stores to the T input and the data with all the stores (treatment and control) to the D input
  • Configure the A/B Controls tool:
    • Select store as the treatment unit identifier
    • Select both trend and seasonality as the numeric measure to match control units to treatment units
    • Select 2 for number of control units per treatment unit
    • Make sure the box is checked to match each control unit to only one treatment unit
  • Repeat steps 3-6 for each region

Workflow Part 2

Preparing Control Treatment Units Part 3

Workflow Steps Part 3

Workflow Steps Part 3

  • Use the Union tool to combine the data from all the regions (no configuration of the tool is needed since all data streams are the same)
  • Run the workflow
  • Use the Join tool to join the output of the union with the store information data (join the treatment field from the union to the store field from the store data, and deselect the right store field to avoid duplicates)
  • Run the workflow, then add an Output tool to the canvas and save the file for later use
