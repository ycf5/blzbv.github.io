17. Performing the Analysis

Performing the analysis

Workflow Steps

Workflow Steps

  • Use the Input tool to import the control-treatment pairs data and the store sales analysis data into a new Alteryx workflow
  • Use the Filter tool to set up a filter for test group = 87.99, so that the true output will stream data for the 87.99 price point test and the false output will stream data for the 76.99 price point test.
  • Add an A/B Analysis tool to the canvas and connect the true output of the filter to the C (control) and T (treatment) inputs
  • Connect the sales data to the P (performance) input of the A/B Analysis tool
  • Configure the A/B Analysis tool
    • In the Controls section of the A/B Analysis - Input Data tab, select controls for the control unit identifier and treatments for the treatment unit mapping field
    • In the Treatments section, select the treatments field as the unit identifier
    • In the Performance Data section, select store as the unit identifier
    • Select week end as the reporting period date
    • Select gross margin as the performance measure
    • Switch to the Test Dates tab and input the test start date (March 20, 2014) and test end date (May 28, 2014)
  • Repeat steps 3-5 to set up an A/B analysis for the 76.99 price test
  • Add a Browse tool to the I output and O output of each A/B analysis
  • Run the workflow to produce HTML (I output) and textual (O output) reports
