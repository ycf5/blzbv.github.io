ARIMA and ETS Automation

Tony referenced the fact that many tools, such as Alteryx, will automate the model selection process. This is incredibly useful as it can save you a lot of time. The default setting on both the ETS and ARIMA tools in Alteryx will automate the selection of the parameters.

However, it's still important to understand the fundamentals of how the models work in order to use them effectively. If you build an ARIMA model using the ARIMA tool that automatically selects terms, then you should still be able to explain what each term means and be able to explain it well to others.
