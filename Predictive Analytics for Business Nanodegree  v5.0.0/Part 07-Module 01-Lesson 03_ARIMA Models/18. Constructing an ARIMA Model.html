18. Constructing an ARIMA Model

Constucting an ARIMA Model

Workflow Steps

Workflow Steps

  • Continuing in the same workflow from the Seasonal Differencing example, connect a “Record ID” tool to the input data
  • Add a “Filter” tool and filter for “record ID <= 114,” to create a holdout sample
  • Connect an “ARIMA” tool to the T output of the filter and configure it:
    • Name the model ARIMA
    • Select “Bookings” as the target field, and set the target field frequency to “monthly”
    • In the “model customization” tab, select “completely user specified model” and configure the terms:
    • Set p to 0 and set d and q to 1
    • Set P to 0 and set D and Q to 1
    • In the “other options” tab, set the series start year to 2006 and the number of periods to include in the forecast to 6
  • Copy-paste the ARIMA tool, give the new one the name “ARIMA_MA2,” and connect it to the T output of the record ID filter
    • In the new tool, change the value of Q to 2
  • Add a “Browse” tool to the R output of each ARIMA tool, and run the workflow
