Workflow Steps

Workflow Steps

  • Use the “Input” tool to import the TB Sales data
  • Connect a “TS Plot” tool to the data stream and configure it:
    • Select “Toothbrush Sales” as the target field
    • Set the target field frequency to “monthly”
    • Set the plot type to “time series plot”
  • Add a “Browse” tool to the I output of the TS Plot tool, and run the workflow
