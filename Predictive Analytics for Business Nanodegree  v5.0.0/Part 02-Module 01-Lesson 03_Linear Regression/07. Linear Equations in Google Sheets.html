07. Linear Equations in Google Sheets

Using Google Sheets To Calculate A Linear Equation

Regression Sheets

Steps for Calculating the Linear Regression

  • Step 1: Open up the data Linear Equation Example Google Sheets.
  • Step 2: Use the Slope function SLOPE(data_y, data_x). Include the values for the target variable in the first field called 'data_y' and the values for the predictor variable in the second field called 'data_x'. The result should be 0.1833.

  • Step 3: Use the Intercept function INTERCEPT(data_y, data_x) using the same values for the same fields. The result should be -11.055.

    • Now, we know the equation of our line:
    Y =0.1833X -11.055
