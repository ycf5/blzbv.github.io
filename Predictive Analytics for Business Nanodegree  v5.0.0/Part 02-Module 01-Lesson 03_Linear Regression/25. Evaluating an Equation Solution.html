25. Evaluating an Equation Solution

Solution

Steps for Evaluating the Equation

  1. Use the coefficients of the linear regression model to determine the expected number of tickets from the new client.

    • Avg Number of tickets = 732(0.1116) + 825000(0.0004858) +12.49 - 18.45
    • Avg Number of tickets = 476.52 so we can assume 477 additional tickets as a result of the new client.

  2. Next, determine the amount of capacity within the firm.

    • With 23 resources that have a capacity of 125 tickets per week
    • The amount of total capacity available is 23*125 = 2875.
    • Therefore, the help desk can handle 2875 tickets per week.

  3. Now, we need to determine if we have enough capacity, or if we need to add resources.

    • We are currently receiving 2800 tickets per week, therefore
    • We currently have excess capacity of 75 tickets per week.
    • The new client will need support for 477 tickets per week, so we don’t have enough capacity to support the new client.
    • We need enough resources for 402 additional tickets per week, or 402/125 = 3.216 resources.

Therefore, we need to add 4 additional resources.
