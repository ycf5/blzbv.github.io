19. Alteryx Walkthrough

Introduction to Alteryx

Alteryx provides analysts with the ability to easily prep, blend, and analyze all of their data using a repeatable workflow, then deploy and share analytics at scale for deeper insights. Analysts can connect to and cleanse data from data warehouses, cloud applications, spreadsheets, and other sources; easily join this data together; then perform analytics – predictive, statistical and spatial – using the same intuitive user interface, without writing any code.

Walkthrough

Below is a quick walkthrough of the Alteryx interface. We also encourage you to go through a few of the tutorials to get a sense for how the program works.

The Tools

Along the top of the interface, you'll see several different icons that are called Tools. You can navigate to different tools by either browsing through the tabs (In/Out, Preparation, Join, etc.) or by typing into the search bar. Once you find the tool you want, you can drag it onto the canvas.

The Configuration Panel

On the left side of the screen, you'll find the configuration panel where you select the configurations you want for each tool. If you need help or want to learn more about the different tools and their respective configurations, you can go to the help section along the top of the Alteryx interface or visit help.alteryx.com.

This particular example shows an input tool. In the configurations panel, you select the data connection, and you can see a sample of the data at the bottom.

The Canvas

The canvas is where you place and connect all the various tools. It's a nice, visual way to see the process the data flows through. You can connect tools by clicking on the output node of one tool, and connecting it to the input node of another. Once you've configured the tools the way you'd like, you run the workflow by clicking on the green arrow icon at the top of the screen. You can then see the results in the results window at the bottom of the interface.

These are just the basics. You'll continue to learn more about Alteryx as you go through the course. Let's keep moving!
