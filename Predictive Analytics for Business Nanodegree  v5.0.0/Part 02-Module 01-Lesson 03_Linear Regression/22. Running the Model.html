22. Running the Model

Running The Model

sTEPs

Steps to run the model and see results

  1. Run the workflow by clicking on the green arrow located above the tool pallet.
  2. Click the OK button to see the results.
  3. Click on the browse tool to see the configuration panel, which will show the output report of the Linear Regression.
  4. To open the report in a new window, click on the drop-down menu at the top of the Configuration panel and select 'New Window (All Records).' Now we can easily read the report in detail.

Reading the results

The second table down, labeled 'Coefficients', has the values that correspond with the b0, b1, and b2 variables in the standard equation we just showed. These are listed in the column labeled 'Estimates', The intercept is the value of b0, and the rest of the coefficients have the labels of the actual variables that we will need to match up with the coefficients.
