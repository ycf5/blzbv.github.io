Getting Familiar with Alteryx

We highly encourage you to spend some time on the Alteryx Tutorials to become more familiar with the tools. You'll learn a lot about Alteryx throughout the program, and this will help you get started.

When you open up Alteryx, the Getting Started window will immediately pop up. You should complete the Preparing Data, Filtering Data, Blending Data, and Analyzing Data tutorials.

If it doesn't pop up you can access by clicking Help in the menu and then by clicking Getting Started
