16. Dummy Variable Quiz

Evaluating an Equation

Let’s do a manual calculation to demonstrate how it works. Let’s start with the following regression equation.

School exp = -468 + (0.067 x avg income) + (1349 x pct under 18) - (14.4 x midwest) - (9.3 x southeast) + (16.5 x west)

Now let’s say we have a state with an average income of 4011, % under 18 of 32.5%, and was in the northeast region. See below for the steps of the calculation.

  1. School exp = -468 + (0.067 x avg income) + (1349 x pct under 18) - (14.4 x midwest) - (9.3 x southeast) + (16.5 x west)
  2. School exp = -468 + (0.067 x 4011) + (1349 x 0.325) - (14.4 x 0) - (9.3 x 0) + (16.5 x 0)
  3. School exp = -468 + 268.737 + 438.425 - 0 - 0 + 0
  4. **School exp = 239.162 **

We’d plug in the numeric values and multiply them by their respective coefficients like this. Then for the region dummy variables, we’d plug in zeros for each of them since the state is in the northeast. Solving the equation results in a predictive school expenditure of about 239.

Interpreting the Dummy Coefficient

One way to explaining the interpretation of the coefficient of one of the dummy variable above is that it represents the average difference between the state expenditures in one region, compared to states in the northeast region, assuming all other variables are held constant. If a state in the midwest has the same pct under 18 and avg income as a state in the northeast, then we’d predict it will spend 14.4 dollars less per student.

Here is our linear regression formula

School exp = -468 + (0.067 x avg income) + (1349 x pct under 18) - (14.4 x midwest) - (9.3 x southeast) + (16.5 x west)

Question 1

What is the predicted expenditures for a state with an average income of 5132, % under 18 of 33.4%, and is in the west region?

Now we plug in these values into the formula to get our predicted expenditures

School exp = -468 + (0.067 x 5132) + (1349 x 0.334) - (14.4 x 0) - (9.3 x 0) + (16.5 x 1)

School exp = -468 + 343.844 + 450.566 - 0 - 0 + 16.5

School exp = 342.91

School exp = 355.545
