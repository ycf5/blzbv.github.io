13. Constructing an ETS Model

Constructing An ETS Model

Workflow Steps

Workflow Steps

  • Use the “Input” tool to import the bookings data
  • Connect a “Record ID” tool
  • Use a “Filter” tool to filter for “record ID <= 114” to create a holdout sample
  • Connect a “TS Plot” tool to the Record ID tool, and configure it:
    • Select “Bookings” as the target field
    • Set the target field frequency to “monthly”
    • Set the plot type to “time series plot”
  • Add a “Browse” tool to the I output of the TS Plot tool, and run the workflow
  • Connect an “ETS” tool to the T output of the filter tool and configure it:
    • Name the model MAM
    • Select “Bookings” as the target field, with monthly frequency
    • In the “model type” tab, set error to “multiplicative,” trend to “additive,” trend dampening to “no,” and seasonal type to “multiplicative”
    • In the “other options” tab, set the number of periods to include to 6, and set the starting period to 2006 period 1
  • Copy-paste the ETS tool and connect the new tool to the T output of the filter tool
  • Name the new model ETS_dampened, and keep all settings the same as the first ETS tool except for changing trend dampening to “yes”
  • Add a “Browse” tool to the R output of each ETS tool, and run the workflow
  • Save the workflow for later use
