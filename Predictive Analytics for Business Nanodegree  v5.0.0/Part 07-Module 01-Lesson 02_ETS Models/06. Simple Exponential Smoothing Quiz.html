06. Simple Exponential Smoothing Quiz

Quiz

Exponential Smoothing Forecasting Quiz

To help understand how exponential smoothing works, let's get a little practice. Using the Single Family Home Sales.xlsx file. Use the exponential smoothing model to forecast the next period of sales using different values for the smoothing parameter.

To calculate the weight in each row, use the formula: Weightt = α * (1 - α)t where α is the smoothing parameter and t is the number of time periods before the most recent period (e.g. t = 0 for the most recent time period, t = 1 for the time period before that).

Remember to use (1-α)n for the very last row as your weight. Refer to the solution file at the bottom of the page if you need to see an example.

The video at the bottom demonstrates the process with a different data set

Using a smoothing parameter (α) of 0.8, what is the forecast for December 1995?
SOLUTION: 46.11

Using a smoothing parameter (α) of 0.6, what is the forecast for December 1995?
SOLUTION: 48.68

SOLUTION: 54.84
