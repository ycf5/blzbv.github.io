Data Preparation Solution - Dealing with Null Values

INSTRUCTOR NOTE:

As a review, there are three main ways to deal with Null values:

  • Filter out records with Null fields
  • Remove the entire field
  • Impute the records

Note on Filter Formula

In this dataset, all records with today day calls with Null values also have today night calls as Null values as well.

Ben uses the expression: !IsNull([total day calls]) AND !IsNull([total night calls]) to filter out rows with fields that have both Null values in total day calls and in total night calls.
