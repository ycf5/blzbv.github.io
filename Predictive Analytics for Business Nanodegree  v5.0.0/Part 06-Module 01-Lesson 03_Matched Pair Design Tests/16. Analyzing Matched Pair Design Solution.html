16. Analyzing Matched Pair Design Solution

Analyzing Match Pair Design Solution

Workflow Steps

Workflow Steps

  • Use the Input tool to import the sales data
  • Use the Filter tool to filter for week ending < January 2, 2011, which will result in showing only data from 2010 up to the week ending on January 1, 2011
  • Use the Formula tool to multiply the weekly customer spend per store by 1.064 to calculate the 6.4% lift and a new value for weekly customer spend
  • Use the Input tool to import the weekly customers per store data
  • Use the Join tool to join the data on store ID and week ending value, then deselect the right values for both columns to avoid duplicates
  • Use the Formula tool to calculate the average spend per customer multiplied by the number of customers
  • Repeat step 6 to calculate the predicted new average spend per customer
  • Calculate the difference between the new and old values for total customer spend
  • Run the workflow
  • Use the Summarize tool to sum the total spend, new total spend, and difference
  • Run the workflow again to see the total sales amount in 2010, the predicted total sales amount had the new product been introduced, and the difference between those values
