13. Analyzing the Results with Alteryx

Analyzing the results with Alteryx

INSTRUCTOR NOTE:

All the files in this example are contained in the zip file Matched Pair Analysis.zip

Workflow Steps

Workflow Steps

  • Use the Input tool to import the control store data, the treatment store data, and the historical performance and experiment data
  • Add an A/B Controls tool to the canvas, and connect the control store data to the C (controls) input, the treatment store data to the T (treatment) input, and the historical and experiment data to the P (performance) input
  • Configure the A/B Controls tool:
    • In the Input Data tab, select controls as the control unit identifier and treatments as the treatment unit mapping field
    • Select store ID as the treatment unit identifier
    • Select right store ID as the performance data unit identifier
    • Select week ending as the reporting period date
    • Select spend customer as the performance measure
    • In the Test Dates tab, enter the experiment start date (October 15, 2011) and end date (March 17, 2012)
  • Right-click on the A/B Controls tool and select add all browses, then run the workflow
