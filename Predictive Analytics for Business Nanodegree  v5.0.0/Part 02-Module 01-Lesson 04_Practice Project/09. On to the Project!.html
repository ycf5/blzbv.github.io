Are you ready?

Now it's time to take on the Project 1! As you work through the project, you have several resources that will help you.

  1. Refer back to the practice project: Project 1 is pretty similar to the practice project, so refer back to the practice project walkthrough if you get stuck.
  2. Refer back to the content: Go back through portions of the course if you need help with a particular topic.
  3. Knowledge: Many others may have the same question you do, and Knowledge can be a great resource.
  4. Study Groups: Other students and mentors are on Study Groups and can help answer questions.

Best of luck!
