02. New Alteryx Tools

New Tools

To help you complete the practice project and project 1, here is an introduction to a couple new tools.

Score Tool

The Score tool is used to apply the results of a predictive model to a dataset. In project zero, you applied the results of a linear regression manually by using the linear regression equation. As you add more predictor variables, manual calculations can become more complicated. The Score tool makes this easy by automatically applying the linear regression equation from a Linear Regression tool.

To use the Score tool, you connect the O (object) node of the Linear Regression tool to one of the input nodes of the Score tool. On the other node, you connect the data you are using to make a prediction. You must make sure that all the predictor variables used in the linear regression model are present in the input dataset used in the Score tool. Generally you can use the default configurations of the Score tool. To read more about it, you can go to the Alteryx Help page here.

The second image above is from the latest version of Alteryx. The Score Tool now requires you to connect the O output of the model to the M input of the score tool and the data you will be applying the model to to the D input.

Once you run the workflow, the resulting dataset will be identical to the input dataset but with one additional field called Score or X. This field represents the predicted values for the target variable in the linear regression model. In the case of the practice project, it's diamond prices. In case of project one, it will be revenue. Formula Tool

The formula tool in Alteryx is similar to the formulas you would use in a spreadsheet application. You can use it to perform various operations on existing fields. For example, if you need to display units in thousands, you can select a field, and put in a formula to divide that field by 1000 for each record.

To use a Formula, you first attach an input dataset. Then, you select from the configurations what record you'd like to use. Lastly, you build the formula. You can do this by either selecting the variables and expressions you'd like to use from the middle section(see below), or you can write the formula manually in the Expression section. Field names must be in brackets. See the example below (which also serves as a hint for how to use the Formula tool in the practice project).

This tool can do a wide variety of things, and you'll learn much more about it in future lessons. To read more about the Formula tool, you can go to the Alteryx Help page:

https://help.alteryx.com/11.0/Reference/Functions.htm.

The second image above shows what the configuration window looks like in the latest version of Alteryx. Notice the top left dropdown underneath Output Column. This field will initially say Select Column. You must click it and select an existing column or select the add column option to create a new one for the formula to output to.

You will find existing fields to reference in your formula by click the X icon on the left. You will find the functions you can include in your formula by clicking the fx icon on the left.
