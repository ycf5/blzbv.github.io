06. Model Comparison Tool Errors

Model Comparison Tool not in predictive tools tab in Alteryx

If you do not see the Model Comparison tool in the predictive tools tab in Alteryx please try the following options:

1) Uninstall Alteryx and reinstall it making sure to select the box to install predictive tools

2) Download the Model Comparison tool directly from the Alteryx: https://gallery.alteryx.com/#!app/Model-Comparison/56bbd3013df7da08b8fcd00a

  • After downloading this file open it and run the workflow in Alteryx
  • Then close Alteryx and restart your computer
  • (Note: This will only work if you are using the latest version of Alteryx

3) If you are using an older version of Alteryx such as 11.7 try using this older version of the model comparison tool file and follow option 2 with it: https://s3.amazonaws.com/video.udacity-data.com/topher/2018/September/5b8dd003_model-comparison/model-comparison.yxzp

4) Consult the Alteryx forums: https://community.alteryx.com/t5/forums/searchpage/tab/message?advanced=false&allow_punctuation=false&inactive=false

Model Comparison Tool Errors

If you get errors using this tool you should check the following.

  1. There should be no null values passed into the models
  2. All Tool Versions should be 1.0 (see images below on how to change Decision Tree and Logistic Regression to 1.0)

Right Click on the Decision Tree Tool, click Choose Tool Version, and then click 1.0

Right Click on the Logistic Regression Tool, click Choose Tool Version, and then click 1.0

If you do not see the ROC Graph

Make sure you have named each model by filling in the Model Name section in the configuration window of each of the models
