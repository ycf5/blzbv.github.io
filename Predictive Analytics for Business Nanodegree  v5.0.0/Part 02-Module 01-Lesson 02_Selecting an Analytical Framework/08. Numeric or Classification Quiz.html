08. Numeric or Classification Quiz

Num vs class

Let’s practice classifying data rich business problems. For each of the following business problems identify whether a numeric or classification model should be used to solve the problem.

Scenario #1

An ice cream vendor sells a different amount of ice cream every week. She puts in an order for ice cream containers every Wednesday, but would like to know how much to order so she has enough ice cream to sell, but doesn’t spend too much on keeping excess inventory frozen.

1

Should a numeric or classification model be used to solve this problem?
SOLUTION: Numeric

2

Scenario #2

A financial organization provides loans to small businesses. Once a pre-loan application is approved, the applicant has 90 days to apply for the actual loan. The financial organization wants to predict how many loan applications they have to process from their pool of pre-loan applicants.

2

Should a numeric or classification model be used to solve this problem?
SOLUTION: Numeric

3

Scenario #3

A marketing organization wants to predict whether someone is likely to redeem a coupon as they would like to minimize costs and only send coupons to people who are likely to use them.

3

Should a numeric or classification model be used to solve this problem?
SOLUTION: Classification
