13. Text: Installing Tableau

Installing Tableau

In the Data Foundations Nanodegree program, Business Analytics Nanodegree program or the Data Analyst Nanodegree program? Then You Can Install Tableau Public - No License Needed

I bet you're pretty excited about Tableau, now you need to install it. You can find it on the Tableau website.

You can use Tableau Public for this course. It's the free version that doesn't connect to as many data sources as the personal or professional versions, but it's more than capable to do most work with. It can read data from text (CSV) files and Excel files which you'll be using in this course. Also, you can't save to workbooks on your computer, but instead save to a public workbook. You can then download the workbook from there, so it isn't much of an issue.

Installing Tableau

Enter your email, then the installer for Tableau Public will download. Follow the installation instructions for your operating system.

Once it's installed, move on to the next part and we'll get started working with Tableau.

Are you in Business Analyst Nanodegree program or Predictive Analytics for Business Nanodegree program?

Installing the Desktop version of Tableau.
